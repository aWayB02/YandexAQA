Тестовое задание: Необходимо разработать пример проекта автотестов для сервиса Яндекс.Диска https://yandex.ru/dev/disk/rest/.
У него есть "полигон" https://yandex.ru/dev/disk/poligon/ и очень подробная документация https://yandex.ru/dev/disk/api/concepts/about-docpage/.
API для тестирования:
https://cloud-api.yandex.net.

Пожалуйста, не используй своего личного пользователя для получения OAuth-токен.

Стек на выбор:
Python3, Pytest (https://github.com/pytest-dev/pytest) и requests (https://github.com/psf/requests).
Java или Kotlin, Junit (https://junit.org/junit5/).
Требования к выполненному проекту:
Проект должен быть размещён на платформе GitHub (https://github.com/) и доступен для просмотра.
Проект должен запускаться локально.
Проект должен включать тесты, проверяющие работу методов GET, POST, PUT и DELETE.
Можно расширить стек, при этом обязательный набор, указанный выше, должен быть включён в каждый из вариантов. Пожалуйста, не используй своего личного пользователя для получения OAuth-токен
